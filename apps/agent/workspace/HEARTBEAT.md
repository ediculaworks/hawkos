# Heartbeat Checklist — Hawk OS

Execute esta checklist a cada execução do heartbeat. Reporte APENAS se houver algo relevante.
Se nada relevante, responda exatamente: HEARTBEAT_OK

## Verificações

### Urgente (sempre checar)
- [ ] Tarefas com due_date hoje ou atrasadas
- [ ] Obrigações legais vencendo nos próximos 3 dias
- [ ] Documentos expirando nos próximos 7 dias

### Diário (se horário 08:00-10:00)
- [ ] Aniversários hoje ou nos próximos 3 dias
- [ ] Eventos do calendário nas próximas 24h
- [ ] Hábitos pendentes do dia anterior
- [ ] Contatos sem interação há mais de 30 dias (top 3)
- [ ] Sono da noite anterior (se logado, comentar qualidade)
- [ ] Medicamentos pendentes de registro

### Semanal (se segunda-feira)
- [ ] Resumo de progresso dos objetivos ativos
- [ ] Hábitos com streak em risco
- [ ] Orçamento: categorias excedidas no mês
- [ ] Padrões de saúde (sono, humor, peso) — tendências da semana

### Mensal (se dia 1)
- [ ] Net worth snapshot
- [ ] Contratos renovando este mês
- [ ] Review de segurança digital

## Formato de Resposta

Se há itens relevantes, responda:
```
**Heartbeat Report**
- [tipo] descrição da ação/alerta
- [tipo] descrição
```

Se nada relevante: HEARTBEAT_OK
